# 같은하늘 (Same Sky)

**같은하늘**은  
지금 같은 장소에 머무는 사람들 사이에  
잠깐 열렸다가 자연스럽게 사라지는 **공개 대화 공간**입니다.

같은 순간, 같은 장소의 맥락을  
부담 없이 나누는 데에만 집중합니다.

---

## ☁️ 하늘

- **하늘 = 하나의 대화 공간**
- 하늘은 장소 기준으로 하나만 열립니다.
- 사람이 있거나 대화의 흔적이 남아 있는 동안만 유지됩니다.
- 사람이 없고, 대화도 사라지면 하늘도 함께 사라집니다.

---

## 💬 대화

- 모든 메시지는 **작성 후 1시간 뒤 자동 삭제**
- 공개 대화만 제공 (DM 없음)
- 한 번에 하나의 하늘에만 참여할 수 있습니다.
